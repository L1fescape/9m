# 9m in NodeJS

> My flight was delayed and I was bored. Inspired by [9m](https://github.com/ehamberg/9m)

## Requirements

- Nodejs
- Mongodb

## Install

```
npm install
```

## Running

Start Mongodb

```
node index.js
```

## Example

```bash
$ curl --data "url=akenn.org" http://localhost:4000/create
```
